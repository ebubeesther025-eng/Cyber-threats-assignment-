# Cyber Threats and Vulnerabilities Assignment

This repository contains my assignment on **cybersecurity threats and vulnerabilities**, with a focus on:
- OWASP ZAP vulnerability scanning
- Phishing Indicators of Compromise (IOC) analysis

---

## 📂 Repository Structure
- **/vulnerability_scan/**  
  Contains OWASP ZAP reports, summaries, and screenshots of detected vulnerabilities.

- **/phishing_analysis/**  
  Includes phishing IOC documentation and case study analysis.

---

## 🛠 Tools Used
- **OWASP ZAP** – for vulnerability scanning and reporting  
- **IOC Analysis** – for identifying phishing-related indicators

---

## 📌 Key Findings
- Common web application vulnerabilities identified during scanning  
- Phishing email patterns and IOC detection strategies  

---

## 🎯 Purpose
This project demonstrates practical skills in identifying and analyzing cybersecurity threats, useful for both academic and professional contexts.
